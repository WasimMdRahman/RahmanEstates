# **App Name**: Rahman Estates

## Core Features:

- Property Listings: Display a list of real estate properties with key details.
- Property Details: Show detailed information about a specific property, including images and features.
- Advanced Search & Filtering: Enable users to search and filter properties based on various criteria.
- Interactive Map: Display properties on an interactive map, allowing users to explore the area.
- Favorite Properties: Allow users to save and manage their favorite properties.
- AI-Powered Property Description Generation: Use a tool to automatically generate compelling property descriptions from listing details.
- Agent Profiles: Display profiles of real estate agents with their contact information and listings.

## Style Guidelines:

- Primary color: Deep slate (#343A40) to convey trust and professionalism.
- Background color: Light gray (#F8F9FA), subtly desaturated, for a clean backdrop.
- Accent color: Muted teal (#66A3A3), adding a touch of modern elegance. Avoid overly bright colors, sticking to the emotional palette described in the prompt.
- Headline font: 'Playfair', serif, for confident, large headlines. This selection is per user request
- Body font: 'PT Sans', sans-serif, chosen because it is highly readable.
- Use modern, clean icons from Lucide Icons to represent property features and amenities.
- Implement a card-based UI with consistent border-radius and an 8pt spacing system.
- Employ subtle animations using Framer Motion for smooth transitions and hover effects.
