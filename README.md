# RAHMAN ESTATES

This is a modern, responsive real estate platform built with Next.js. It allows users to browse property listings, view property details, learn about real estate agents, and more.

Property Listings: Browse a comprehensive list of available properties.
Advanced Filtering: Filter properties by price, type, bedrooms, and location.
Detailed Property Pages: View detailed information, image carousels, and agent contacts for each property.
Agent Profiles: Get to know the real estate agents with dedicated profile pages.
Interactive Map View: Explore properties geographically on an interactive map.
Favorites: Save your favorite properties for easy access.
Responsive Design: Fully responsive layout for a great experience on any device.
Framework: Next.js (with App Router)
Language: TypeScript
UI: React
Styling: Tailwind CSS
UI Components: ShadCN/UI
Icons: Lucide React
